# Opera Shield Pro v2.1

Load in Opera with:
1. Extract the zip.
2. Open `opera://extensions`.
3. Enable Developer mode.
4. Click Load unpacked.
5. Select this folder.

Included in this package:
- request inspector
- per-rule hit counts
- importable custom subscription URLs
- element picker UI
- refined popup blocking that targets remote ad-like popups instead of every popup
- overflow-safe logger/popup rows
- donation button and PayPal QR code
