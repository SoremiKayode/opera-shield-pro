Quick Help

Install in Opera (manual package)

  1. Click the one-click download button to download the zip package.
  2. Unzip the package to a folder on your device.
  3. Open opera://extensions and enable Developer mode.
  4. Click Load unpacked and select the unzipped folder.

Troubleshooting basics

  • Refresh filter lists from your plugin popup panel.
  • Temporarily disable protection for one domain to test breakage.
  • Reload the page after changing blocking settings.
