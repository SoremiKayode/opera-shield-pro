## Suggested long description
Opera Shield Pro helps you browse faster and cleaner by reducing ad, tracking, and nuisance requests. Built for users who want strong defaults plus transparent controls, it combines practical protection with easy troubleshooting.

### Why users install Opera Shield Pro
- Faster page loads on ad-heavy sites
- Fewer visual distractions and intrusive popups
- Better privacy through request-level filtering
- Clear controls for per-site behavior

### Core features
- Network request filtering with customizable lists
- Popup-focused nuisance blocking
- Element picker for on-page cleanup
- Request inspector and logger for troubleshooting
- Rule hit counters and importable custom subscription URLs

### Designed for trust
- No account required
- Local settings storage in browser extension storage
- Clear support and privacy policy pages
